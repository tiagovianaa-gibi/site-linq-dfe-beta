# Instagrams das Quadrilhas

**VOCÊ SÓ MUDA AQUI**: Adicione os Instagrams das quadrilhas no arquivo `data/quadrilhas.json`

## Formato

No campo `instagram` de cada quadrilha, adicione o handle com @:

```json
{
  "id": 1,
  "nome": "Amor Junino",
  "instagram": "@amorjunino"  ← Adicione aqui
}
```

## Como adicionar

1. Abra `data/quadrilhas.json`
2. Encontre a quadrilha pelo `id` ou `nome`
3. Adicione o Instagram no campo `instagram` (ex: "@nome_da_quadrilha")
4. Salve o arquivo

O site automaticamente criará links clicáveis para o Instagram quando o campo estiver preenchido.
